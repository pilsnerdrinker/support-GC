# P4グランドクロス補助表

FF14「絶妖星乱舞」P4 グランドクロスのデバフ・真偽・カオスを入力して、自分がどう動くかを確認する補助表です。

公開ページ: https://pilsnerdrinker.github.io/practice-grandcross-helper/

## 使い方

1. `1回目GC` タブで、1回目の情報を入力します。
2. `2回目GC` タブで、2回目の情報を入力します。
3. `結果` タブで、1回目処理・2回目処理の行動を確認します。

## 入力するもの

### 1回目GC

- エクスデス真偽
- ついたデバフ
  - 水
  - 雷
  - 視線
  - 加速のみ
- 水雷がついた場合のみ、水雷の早遅
- カオス真偽
- ほのお / みず

### 2回目GC

- エクスデス真偽
- ついたデバフ
  - 水
  - 雷
  - 視線
  - 加速のみ
- 水雷がついた場合のみ、水雷の早遅
- カオス真偽

2回目のほのお/みずは、1回目の入力から自動で決まります。

## 結果の見方

結果タブは、実際に見る処理タイミングに合わせて2枠だけ表示します。

- `1回目処理（早デバフ）`
- `2回目処理（遅デバフ）`

該当しない行は表示しません。

表示される行だけ見て動いてください。

## デバフの法則

### 視線

- GC1の視線は必ず早視線
- GC2の視線は必ず遅視線

### 水雷

- 同じGC回の水雷は必ず同じ早遅
- GC1の水雷が早なら、GC2の水雷は遅
- GC1の水雷が遅なら、GC2の水雷は早

### ほのお/みず

- ほのおは必ず早い
- みずは必ず遅い

### 加速弾

- 加速弾は必ず視線の2人につく
- 加速弾は、同じGC回の水雷4人にはつかない
- 加速弾のみ付与される人が2人いる
- GC1の視線は早加速弾
- GC1の加速のみは遅加速弾
- GC2の視線は遅加速弾
- GC2の加速のみは早加速弾

## 入力チェック

結果タブ下部に、未入力や法則上おかしい組み合わせを表示します。

- 水/雷を選んで早遅が未入力
- 1回目と2回目が両方とも水雷側
- 1回目と2回目が両方とも視線/加速側
