# P4グランドクロス補助表

FF14「絶妖星乱舞」P4 グランドクロス用の、真偽・早遅・床処理をその場で確認するための補助ページです。

## 概要

グランドクロス中に確認した情報をセルで入力すると、対応する行動を表内に即表示します。

主な目的は、デバフ欄やチャット欄の情報を見たあとに、

- 早い/遅いのどちらで処理するか
- 集合/散開
- 視線を見る/見ない
- 加速度で止まる/動く
- ほのお/つなみの処理
- マジックチャージの雷床/氷床

をすばやく確認できるようにすることです。

## 使い方

### 1回目 / 2回目

「1回目」「2回目」の表で、エクスデスの真偽を選びます。

- 真：水色の〇
- 偽：赤い●＋黄色い？

水・雷・加速度は、該当するデバフの早遅に合わせて押します。

- 早デバフが付いた場合：1回目の行を押す
- 遅デバフが付いた場合：2回目の行を押す

押すと、その行の真偽に応じた行動が表示されます。

### 視線

視線は手入力ではなく、以下の前提で自動表示します。

- 1回目：早視線
- 2回目：遅視線

1回目/2回目の真偽を選ぶと、「見る」「見ない」が自動で表示されます。

### ほのお / つなみ

ほのお・つなみはそれぞれ独立した表です。

各表で真/偽を押すと、対応する行動が表示されます。

### マジックチャージ

マジックチャージは雷床・氷床を別々に入力します。

- 雷：斜めストライプの床アイコン
- 氷：十字4分割の床アイコン

真/偽を押すと、踏む/踏まないを表示します。

## 表示仕様

### 真偽

- 真：水色の〇、選択時は水色背景
- 偽：赤い●＋黄色い？、選択時は薄い赤背景

### 加速度

- 真：止まる、赤背景
- 偽：動く、緑背景

### 視線

- 見る/見ないをピンク背景で表示

### 水・雷

- 頭割り/散開を表示

## スマホ表示

スマホで開いたときは、できるだけ一画面に収まるように余白・文字サイズ・アイコンサイズを圧縮します。

画面が小さい場合は、タイトルや説明を省略して表を優先表示します。


## 注意

この補助ページは、手入力した内容をもとに行動を表示する確認用ページです。

攻略法・処理方法の変更や固定/野良のローカルルールによって、表示内容と実際の処理が異なる場合があります。使用前に、参加するパーティーの処理と合っているか確認してください。
